# Kusikusi Frontend

> The generic CMS frontend that can be used as a base for other Kusikusi based frontends

## Running in development mode

###### run development server (with default theme)
```bash 
$ quasar dev
```

###### PWA
```bash 
$ quasar dev -m pwa
```

###### Electron App
```bash 
$ quasar dev -m electron
```
